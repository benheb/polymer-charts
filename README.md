### Developing:

Install dependencies:

```bash
npm install
```

```bash
bower install
```

Start local server:

```bash
grunt serve
```

Deploy:

```bash
git checkout gh-pages
```

```bash
grunt build
```

```bash
grunt gh-pages
```
